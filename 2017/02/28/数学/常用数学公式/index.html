平方和与立方和

设\(m, n\)为正整数，前\(n\)个正整数的\(m\)次幂的和满足公式：
$$1^m+2^m+…+n^m=a_1n+a_2n^2+…+a_{m+1}n^{m+1}$$
当\(m=1\)时，\(1+2+…+n=\frac{n(n+1)}{2}\)
当\(m=2\)时，\(1^2+2^2+…+n^2=\frac{n(n+1)(2n+1)}{6}\)
当\(m=3\)时，\(1^3+2^3+…+n^3=\frac{n^2(n+1)^2}{4}\)
