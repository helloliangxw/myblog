随机试验

自然界与社会生活中的两类现象：

  • 确定性现象
    在一定条件下必然发生的现象。
    如：在一个标准大气压下，水加热到100℃一定会沸腾；向上抛出的物体会落下
  • 随机现象
    在一定条件下具有多种可能结果， 且试验时无法预知出现哪个结果的现象。
    如：掷骰子可能出现“1点”，也可能是其他情况；检验产品可能是合格品，也可能是不合格品

对随机现象的观察、记录、实验统称为随机试验。
它具有以下特性：

  • 可以在相同条件下重复进行
  • 事先知道所有可能出现的结果
  • 进行试验前并不知道哪个试验结果会发生

例如：

  1. 抛一枚硬币，观察试验结果
  2. 对某路公交车某停靠站登记下车人数
  3. 对听课人数进行一次登记

样本空间、随机事件

随机试验的所有可能结果构成的集合称为样本空间，记为\(S=\left \{ e \right \}\)，\(S\)中的元素\(e\)称为样本点。

例如：

  • 一枚硬币抛一次
    \(S=\left \{ 正面, 反面 \right \}\)
  • 记录一城市一日中发生交通事故次数
    \(S=\left \{ 0, 1, 2, … \right \}\)
  • 记录一批产品的寿命
    \(S=\left \{ x:x\geqslant 0 \right \}\)
  • 记录某地一昼夜最高温度x，最低温度y
    \(S=\left \{ a \leqslant y \leqslant x \leqslant b \right \}\)

样本空间\(S\)的子集\(A\)称为随机事件\(A\)，简称：事件\(A\)。
当且仅当\(A\)中的某个样本点发生称事件\(A\)发生。
事件\(A\)的表示可用集合，也可用语言来表示。

例如：观察某公交站的候车人数，样本空间\(S= \left \{ 0, 1, 2, … \right \}\)
事件\(A\)表示“至少有5人候车”， \(A=\left \{ 5, 6, 7, … \right \}\)
事件\(B\)表示“候车人数不多于2人”， \(B = \left \{ 0, 1, 2 \right \}\)

如果把\(S\)看作事件，则每次试验\(S\)总是发生，所以\(S\)称为必然事件。
如果事件只含有一个样本点，称其为基本事件。
如果事件是空集，里面不包含任何样本点，记为\(\phi \)，则每次试验\(\phi \)都不发生, 称\(\phi \)为不可能事件。

例如：观察某公交站的候车人数，样本空间\(S= \left \{ 0, 1, 2, … \right \} \)。
事件\(C\)表示“恰好有3人侯车”,\(C= \left \{ 3 \right \}\)是基本事件；
事件\(D\)表示“候车人数既少于3个又多于3”,\(D=\phi \)是不可能事件。

事件的相互关系及运算

事件的关系

  • \(A \subset B\)
    事件\(A\)发生一定导致\(B\)发生
  • \(A = B\)
    即\(A \subset B\)且\(B \subset A\)

事件的运算

  • 和事件
    \(A\)与\(B\)至少有一个发生
    \(A \cup B=\left \{ x:x \in A 或 x \in B \right \}\)
  • 积事件
    \(A\)与\(B\)同时发生
    \(A \cap B=\left \{ x:x \in A 且 x \in B \right \}\)
  • 差事件
    \(A-B=\left \{ x:x \in A 且 x \notin B \right \}\)
    \(A-B=A\bar{B}=A \cup B-B=A-AB\)
  • 逆事件
    \( \bar{A}\)也称\(A\)的互逆、对立事件
    \( A \cup \bar{A}=S, A\bar{A}=\phi, \bar{\bar{A}}=A \)
  • 不相容或互斥事件
    当\( AB=\phi \)时，称\( A\)与\(B\)不相容

运算定律

  • 交换率
    \(A \cup B=B \cup A\)
    \(A \cap B=B \cap A\)
  • 结合律
    \(A \cup (B \cup C)=(A \cup B) \cup C\)
    \(A \cap (B \cap C)=(A \cap B) \cap C\)
  • 分配率
    \(A \cup (B \cap C)=(A \cup B) \cap (A \cup C)\)
    \(A \cap (B \cup C)=(A \cap B) \cup (A \cap C)\)
  • 对偶率（得摩根定律）
    \(\overline{A \cup B}=\bar{A} \cap \bar{B}\)
    \(\overline{A \cap B}=\bar{A} \cup \bar{B}\)
    推广：
    \(\overline{A \cup B \cup C}=\bar{A} \cap \bar{B} \cap \bar{C}\)
    \(\overline{A \cap B \cap C}=\bar{A} \cup \bar{B} \cup \bar{C}\)
    …

注意：\(\overline{AB}\)与\(\bar{A}\bar{B}\)的区别。
\(\overline{AB}\)表示\(A和B\)不同时发生，\(\bar{A}\bar{B}\)表示\(A和B\)都不发生。

上图为\(\overline{AB}\)，下图为\(\bar{A}\bar{B}\)。
两者的关系为：\(\overline{AB}=\bar{A}\bar{B} \cup A\bar{B}\cup \bar{A}B\)

例:用\(A, B, C\)三个事件关系及运算表示下列各事件:

黄色：三者都不发生；橙色：恰有一个发生；白色：恰有两个发生；红色：三者都发生。

  • \(A\)发生\(B, C\)都不发生
    \(A\bar{B}\bar{C}=A-B-C\)
  • 三者同时发生
    \(ABC\)
  • 三者都不发生
    \(\bar{A}\bar{B}\bar{C}\)
  • 恰有一个发生
    \(A\bar{B}\bar{C} \cup \bar{A}B\bar{C} \cup \bar{A}\bar{B}C\)
  • 恰有两个发生
    \(\bar{A}BC \cup A\bar{B}C \cup AB\bar{C}\)
  • 至少有一个发生
    \(A\cup B \cup C=S-\bar{A}\bar{B}\bar{C}=\overline{\bar{A}\bar{B}\bar{C}}=(A\bar{B}\bar{C} \cup \bar{A}B\bar{C} \cup \bar{A}\bar{B}C)\cup (\bar{A}BC \cup A\bar{B}C \cup AB\bar{C}) \cup ABC\)
  • 至少有两个发生
    \(AB \cup AC \cup BC=(\bar{A}BC \cup A\bar{B}C \cup AB\bar{C}) \cup ABC\)
  • 至少有一个不发生
    \(\bar{A} \cup \bar{B} \cup \bar{C}=\overline{ABC}=(A\bar{B}\bar{C} \cup \bar{A}B\bar{C} \cup \bar{A}\bar{B}C) \cup (\bar{A}BC \cup A\bar{B}C \cup AB\bar{C}) \cup \bar{A} \bar{B} \bar{C}\)

频率

频率是0～1之间的一个实数，在大量重复试验的基础上给出了随机事件发生可能性的估计。
定义：$$f_{n}(A)=\frac{n_A}{n}$$
其中\(n_A\)是\(A\)发生的次数，即频数；\(n\)是试验总次数。
称\(f_{n}(A)\)是\(A\)在这\(n\)次试验中的频率。

频率的性质：

  1. \(0 \leqslant f_{n}(A) \leqslant 1\)
  2. \(f_{n}(S)\)=1
  3. 若\(A_1, A_2,…,A_k \)互不相容，则$$f_{n}(\bigcup_{i=1}^{k}A_i)=\sum_{i=1}^{k}f_{n}(A_i)$$
  4. \(f_{n}(A)\)随\(n\)的增大逐渐稳定，稳定值为\(p\)，即为概率

概率

概率的统计性定义：
当试验的次数增加时，随机事件\(A\)发生的频率的稳定值\(p\)称为概率.记为\(P(A)=p\)

概率的公理化定义：
设随机试验对应的样本空间为\(S\)，对每个事件\(A\)，定义\(P(A)\)，满足：

  1. 非负性：\(P(A) \geqslant 0\)
  2. 规范性：\(P(S) =1\)
  3. 可列可加性：若\(A_1, A_2,… \)互不相容，则$$P(\bigcup_{i=1}^{\infty}A_i)=\sum_{i=1}^{\infty}P(A_i)$$

称\(P(A)\)为事件\(A\)的概率。

概率的性质：

  1. \(P(\phi)=0\)
  2. \(P(A)=1-P(\bar{A})\)
  3. 有限可加性：若\(A_1, A_2,…A_k \)互不相容，则$$P(\bigcup_{i=1}^{k}A_i)=\sum_{i=1}^{k}P(A_i)$$
  4. 若\(A \subset B\)，则\(P(B-A)=P(B)-P(A)\)
  5. 加法公式：\(P(A \cup B)=P(A)+P(B)-P(AB)\)

\(P(A \cup B \cup C)=P(A)+P(B)+P(C)-P(AB)-P(AC)-P(BC)+P(ABC)\)
证：\(P(A \cup B \cup C)\\=P(A\cup B)+P(C)-P(AC \cup BC)\\=P(A)+P(B)-P(AB)+P(C)-(P(AC)+P(BC)-P(ABC))\)
