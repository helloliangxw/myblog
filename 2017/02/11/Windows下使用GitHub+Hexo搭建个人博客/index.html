创建GitHub仓库

  1. 注册GtiHub
  2. 新建公共仓库，仓库命名规则为：*.github.io。以我的为例：myblog.github.io

环境准备

安装Git

下载地址：GitHub for windows

安装完成后包括GitHub客户端和GitShell：

  1. 打开GitHub客户端，输入账户名和密码，登录GitHub
  2. 在本地新建GitHub目录（可以是任意位置），将之前创建的远程仓库clone到该目录下（如：E:\MyDocuments\GitHub\myblog.github.io），即为本地仓库

PS：无需配置SSH密钥

安装Node.js

下载地址：Node.js for windows

下载后直接安装即可

部署本地博客

  1. 在GitHub下新建hexo目录
  2. 打开GitShell，定位到hexo目录。输入hexo init，对hexo进行初始化。运行结束后，hexo文件夹下会生成一系列的文件。
  3. 将第2步中hexo文件夹下生成的所有的目录和文件拷贝至本地仓库

至此，本地hexo博客已经搭建好了。

本地博客查看方式：

打开http://localhost:4000/，即可看到本地博客。

将独立域名与GitHub空间绑定

方法一：在远程仓库的根目录下面，新建一个名为CNAME的文本文件，里面写入要绑定的域名，比如helloliangxw.com。
方法二：将CNAME文件放进本地仓库source目录下，这样每次使用hexo g -d命令后，hexo都会将CNAME文件提交至远程仓库根目录。同样可以将README文件放在source目录下。

如此，就可以通过域名访问博客。
