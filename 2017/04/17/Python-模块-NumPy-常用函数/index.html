梯度函数

函数 说明
np.gradient(f) 计算数组f中元素的梯度，当f为多维时，返回每个维度梯度

梯度：连续值之间的变化率，即斜率。
XY坐标系中，连续三个X坐标对应的Y轴值为a、b、c，那么b的梯度是：(c‐a)/2

对于[a, b, c, d]而言，
a的梯度为：(b-a)/1
b的梯度为：(c-a)/2
c的梯度为：(d-b)/2
d的梯度为：(d-c)/1
