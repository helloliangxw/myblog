递归

函数定义中使用函数自身的方法。
经典例子：阶乘。

$$n!=n(n-1)(n-2)…1=\left{\begin{matrix}
1 && n=0 \\
n(n-1)! && otherwrise
\end{matrix}\right.$$

其中\(n=0\)称为该递归的基例。

递归特征：

  1. 递归不是循环
  2. （一个或多个）基例不需要递归，否则递归就会无限循环
  3. 所有的递归链都要以一个基例结尾

PS：Python默认“递归深度的最大值”为900，达到900次调用时，程序就会终止。
